# Legacy LogGenius Scripts

This folder keeps older LogGenius, DouBao/GPT augmentation, low-diversity, and oracle-generation scripts.

They are not the entry points for the final parser-diversity experiment. Some scripts refer to historical folders such as `log_after_gpt`, `log_after_entropy`, `log_after_group`, `logs`, or `Dataset`, which are not part of the cleaned final project layout.

Use the top-level scripts in `Code/` for the final reproducible experiment.
